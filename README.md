# Popular Movies
Required Project for the Associate Android Developer Fast Track Nanodegree Program on Udacity.

This project uses https://api.themoviedb.org api to fetch popular and top rated movies depending on user preferecence.
An Api key is required to access this api. To use this project, please grab an api key for free at https://themoviedb.org 
and insert on line 39 of the MainScreen.java file in the app directory of the project.
